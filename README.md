# Cyber Portal 🌐

A cyberpunk-style personal portal dashboard with stunning visual effects.

## Features

- 🎨 **Cyberpunk Design** - Neon colors, glitch effects, scanning lines
- ⚡ **Particle Background** - Interactive particle system with mouse tracking
- 🕐 **Neon Clock** - Real-time digital clock with binary display
- 📊 **System Stats** - Animated wave visualizations and simulated metrics
- 🔗 **Quick Links** - Fast access to your favorite sites
- 💬 **Quote Card** - Random quotes with terminal-style typing effect
- 🐙 **GitHub Card** - Contribution graph and stats

## Tech Stack

- Vue 3 + Vite
- Pure CSS animations (no external libraries)
- GitHub Pages deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push to `main` branch, GitHub Actions will auto-deploy to GitHub Pages.
